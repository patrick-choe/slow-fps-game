#### A very slow fps game in spigot 1.12.2

> ---
> #### Features
> - Breaks the common sense that fps should be fast.
> - The projectile speed is super slow.
> - Can use any item as the projectile.
> ---
> #### Gradle (Groovy)
>```groovy
>allprojects {
>    repositories {

>    }
>}
